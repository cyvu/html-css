Forms

This section will have some forms to show off their default structure and style.

The "name" attribute should always be applied to any element that sends any data (including buttons, hidden inputs).

The "id" is needed to associate elements to their label.

The "autofocus" attribute moves the focus to this element once the page has loaded. Useful for a login form.

Label

Labels should always be used for input types of text, checkbox, radio, file and password
